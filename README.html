Panduan Instalasi Sistem Informasi Masjid

Masjid Al-Ikhlas (Django Project)

Prasyarat: Pastikan Python 3.8+ dan pip sudah terinstal di komputer Anda.

1Persiapan Lingkungan (Virtual Environment)

Disarankan menggunakan virtual environment agar dependensi proyek tidak tercampur dengan sistem global.

# Windows
python -m venv venv
venv\Scripts\activate

# macOS / Linux
python3 -m venv venv
source venv/bin/activate

2Instalasi Dependensi

Instal paket-paket Python yang dibutuhkan oleh aplikasi ini.

pip install django django-crispy-forms crispy-bootstrap5 pillow adhanpy pytz currency-symbols
*Jika ada file requirements.txt, gunakan: pip install -r requirements.txt

3Setup Database & Migrasi

Inisialisasi database SQLite (default) dan jalankan migrasi tabel.

python manage.py makemigrations
python manage.py migrate

4Import Dummy Data (Data Contoh)

Aplikasi ini menyediakan perintah khusus untuk mengisi database dengan data contoh (Jadwal Sholat, Berita, Kegiatan, Keuangan, Donatur, dan Akun Admin).

python manage.py populate_dummy_data
Akun yang Dibuat Otomatis:
  • Superuser: Username: admin / Password: admin
  • Staf: Username: staff / Password: staff

5Menjalankan Server

Jalankan server pengembangan lokal.

python manage.py runserver

6Akses Aplikasi

  • Halaman Publik http://localhost:8000
  • Panel Admin (Dashboard Pengurus) http://localhost:8000/panel/
  • Display Digital Signage (TV Masjid) http://localhost:8000/display/
  • Django Admin (Superuser) http://localhost:8000/admin/
Fitur Utama:
  • Jadwal Sholat Otomatis (Lokasi & Koordinat)
  • Koreksi Waktu Sholat
  • Manajemen Keuangan (Pemasukan/Pengeluaran)
  • Laporan Kas Bulanan/Tahunan
  • Manajemen Donatur Tetap
  • Manajemen Kegiatan & Berita
  • Digital Signage View (Mode Layar Penuh)
  • Panel Admin Responsif
